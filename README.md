# Valogen

Simple 3D engine with Vulkan API

Features
- [ ] Window system
- [ ] Shader class
- [ ] Textures
- [ ] Camera
- [ ] 3D
- [ ] Mapping
- [ ] Models loading
- [ ] Text
- [ ] GUI

## Installation
1. Run ``compile.sh``
2. Create ```build``` directory and run `cmake ..` from `build` directory 
3. Run ``make``
